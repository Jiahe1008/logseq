- | 格式字符 | 含义 | 示例 |
  | ---- | ---- | ---- |
  | `x` | 十六进制 | `x/4x 0x401000` |
  | `d` | 十进制 | `x/4d 0x401000` |
  | `u` | 无符号十进制 | `x/4u 0x401000` |
  | `o` | 八进制 | `x/4o 0x401000` |
  | `t` | 二进制 | `x/4t 0x401000` |
  | `c` | 字符 | `x/16c 0x401000` |
  | `s` | 字符串 | `x/s 0x401000` |
  | `i` | 指令 | `x/10i 0x401000` |
  | `a` | 地址 | `x/4a 0x401000` |
  | `f` | 浮点数 | `x/4f 0x401000` |

  | ---- | ---- | ---- |
  | `b` | 字节 | `x/4xb` |
  | `h` | 半字（2字节） | `x/4xh` |

  | `g` | 双字（8字节） | `x/4xg` |
