	- OF的置0逻辑：
		- ### 1. 符号位判断逻辑（直观理解）
		  
		  这是最常用的逻辑，适用于加法和减法：
		- **加法 ($A + B$)**：
			- 如果两个**符号相同**的数相加，结果的**符号却与它们相反**，则 $OF = 1$。
			- **正 + 正 = 负** $\rightarrow$ 溢出
			- **负 + 负 = 正** $\rightarrow$ 溢出
			- **正 + 负** $\rightarrow$ **永远不会溢出**（因为结果的绝对值一定比原操作数小，不可能超出范围）。
		- **减法 ($A - B$)**：
			- 减法被视为 $A + (-B)$。
			- 如果**符号不同**的两个数相减，结果的**符号与被减数 ($A$) 相反**，则 $OF = 1$。
			- **正 - 负 = 负** $\rightarrow$ 溢出
			- **负 - 正 = 正** $\rightarrow$ 溢出
			- **符号相同**的两个数相减 $\rightarrow$ **永远不会溢出**。
